# Khati Bazaar Admin Panel

এটি একটি অ্যাডমিন প্যানেল প্রজেক্ট যা 'খাটি বাজার' (Khati Bazaar) এর জন্য তৈরি করা হয়েছে।

## প্রজেক্ট সম্পর্কে:
এই অ্যাডমিন প্যানেলটি ব্যবহার করে এডমিনরা অর্ডার এবং সেলারদের তথ্য ম্যানেজ করতে পারবেন।

## কীভাবে শুরু করবেন (How to run):
১. প্রজেক্টটি ক্লোন করুন।
২. `npm install` কমান্ড দিন।
৩. `npm start` দিয়ে প্রজেক্টটি রান করুন।
